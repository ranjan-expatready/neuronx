# Architecture Decision Records (ADR)

**Purpose**: Document significant architectural and governance decisions for the Autonomous Engineering OS.

**Format**: Each decision is recorded in a separate file following the naming convention:

```
NNNN-title-in-kebab-case.md
```

Where `NNNN` is a zero-padded sequence number.

2. Fill in all sections
3. Submit via PR following standard governance
4. Update this README's Decision Index
